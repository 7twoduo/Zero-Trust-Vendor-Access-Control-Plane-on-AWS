11. Services you will build

These are your actual application services.

1. Access Request API

Receives requests for access.

2. Policy Engine

Checks whether requested access fits policy.

3. Approval Service

Routes request for approval.

4. Access Broker

Issues temporary AWS access or approved session path.

5. Evidence Service

Stores approval and access evidence.

6. Exception Service

Handles policy exceptions and compensating controls.

7. Findings Ingestion Service

Pulls in GuardDuty, Security Hub, Config, Inspector, Access Analyzer.

8. Admin Portal API

Provides secure views for admins and security staff.

9. Reporting Worker

Generates control and audit reports.
